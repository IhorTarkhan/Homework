Data structure:
    Coordinates of points in three-dimensional space (valid), first at the distance from the origin, in ascending order.

Implement sorting algorithms for a given data structure (list 1). Implement the following algorithms:
    1) Insertion of varieties;
    2) Quicksort (Hoare partition scheme, leftmost pivot);
    3) Merge varieties (bottom-up);
    4) Combined algorithm that uses a more efficient algorithm (one of quicksort or merge sort) for large arrays and a
    simple algorithm for small arrays;
    5) Sorting using library functions.

The following operating modes must be implemented:
    1. Demonstration mode - sets the size of the array, generates an array of the appropriate size, the array is sorted
    by different algorithms, outputs intermediate steps of each algorithm;
    2. Automatic benchmark mode with speed measurements of each algorithm.

All algorithms should work on the same arrays. Consider:
    - random arrays;
    - almost sorted in the correct order;
    - almost sorted in the wrong order.
