Implement the data structure "ordered list" based on:
  1. Linked List
  2. Array List
  3. Binary search tree (without balancing)
  4. AVL trees
  5. 2-3 trees

Implement operations: create an empty list, add an element, delete an element, search for elements by value and range of
values, display elements in the correct order, perform certain actions on all elements in the correct order. Also
implement filling the list with a given number of random items.

It is necessary to implement the following modes of operation:
  1. Interactive dialog mode - when the user can choose which operations to perform and set the required parameters.
  2. Demonstration mode - set fixed values ​​and sequences of operations that demonstrate the correct operation of data
structures and operations in different situations. The user does not have to enter anything in this mode.
  3. Automatic “benchmark” mode with measurements of used memory and speed of each implementation.

Data type:
  7. Coordinates of points in three-dimensional space (valid), first by the distance from the origin, by ascending
